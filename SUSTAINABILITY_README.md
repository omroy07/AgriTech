# Sustainable Farming Eco-Toolbox

This feature provides interactive tools to help farmers manage organic resources and reduce environmental impact.

## Tools Included:
1. **Compost Tracker**: A simulation tool to understand the 90-day biological breakdown phases of compost.
2. **Pesticide Finder**: A searchable database of DIY organic pesticide recipes.
3. **Water Calculator**: A mathematical tool to estimate water savings when transitioning to efficient irrigation.

## Integration:
This feature is modular. To add it to the main navigation, link to `eco_toolbox.html`.
